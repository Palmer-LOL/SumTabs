Privacy & Permissions

Tab access: SumTabs needs access to tab URLs in order to group tabs by domain. All URL processing happens locally inside your browser. No tab data is transmitted anywhere.

Storage: SumTabs stores its settings using Chrome’s extension storage. If Chrome Sync is enabled in your browser, these settings may sync across your devices as part of your browser profile. If Sync is disabled, settings remain local to this device.

The extension does not send data to any external server and does not include analytics, telemetry, or remote code. The source code is available for review.
